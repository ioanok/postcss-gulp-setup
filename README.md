## [PostCSS-Gulp-Setup]

### Quick start

##### - install node.js and NPM on your machine

##### - clone the repository:    
```
git clone https://github.com/ioanok/postcss-gulp-setup
```
##### - install dependencies by running 
```
cd postcss
npm install // to install the dependencies from package.json
``` 
##### - run gulp css
```
gulp css 
```
